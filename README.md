# 🎓 InterviewHelp AI

An AI-powered interview preparation assistant that generates personalized technical, HR, and coding interview questions based on a student's skills, target domain, and experience level.

Built using **Python**, **Streamlit**, **LangChain**, and **Groq LLM**.

---

## 🚀 Features

- 📝 Personalized interview preparation
- 💻 Technical interview questions with model answers
- 💼 HR interview questions with sample answers
- 👨‍💻 Coding interview questions with:
  - Difficulty level
  - Expected approach
  - Key concepts
  - Time complexity (where applicable)
- 📚 Recommended topics to prepare next
- 🎯 Personalized improvement suggestions
- ⭐ Interview readiness score
- 🗓️ 30-Day interview preparation roadmap
- 🎨 Clean and responsive Streamlit interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/InterviewHelp-AI.git
cd InterviewHelp-AI
```

### 2. Create a virtual environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### macOS/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```env
GROQ_API_KEY=your_groq_api_key
```

Get your free API key from:

https://console.groq.com/keys

### 5. Run the application

```bash
streamlit run app.py
```

## 📖 How It Works

1. Enter your technical skills.
2. Select your target domain.
3. Choose your experience level.
4. Click **Generate Response**.
5. Receive a complete interview preparation guide generated by AI.
